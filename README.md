Online Electronic Strore

Developed a system to streamline the billing and inventory management processes for an electronic store, with distinct user roles for admin and employee. Enabled admins to manage inventory updates, employee credentials, and overall store operations, enhancing control and oversight.

Designed functionality for employees to handle billing, track product issuance, and manage returns, including automated late return fines and reminder notifications. Improved operational efficiency by integrating real-time inventory tracking and automated billing, resulting in a smoother workflow for both inventory management and customer transactions
